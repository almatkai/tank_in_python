<h2>How to run</h2>
<p>Just run the main.exe file</p>

<h2>Controllers</h2>
<h3>Main Controllers</h3>
<p>P- pause</p>
<p>Space- restart(Only if the game is Paused and if the game is over)</p>
<h3>Player1</h3>
<p>DEFAULT WASD to move</p>
<p>E- shoot</p>
<h3>Player2</h3>
<p>DEFAULT arrows ←↑→↓ to move</p>
<p>Right CTRL- shoot</p>
